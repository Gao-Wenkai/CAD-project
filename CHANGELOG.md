# 更新日志 — Engineering CAD v1.1

> 对比上一版本 (`fd520b4`：鼠标点击绘制直线、多段线、圆、弧)

---

## 新增功能

### 1. AutoCAD 风格交互式命令行
- **高度** 从 24px 增至 **60px**（扩大 2.5 倍）
- **字体** 从 16pt 升级至 **20pt Consolas** 等宽字体
- **深色背景** `#28282C` 配白色文字 `#F0F0F0` — 与 AutoCAD 外观一致
- **多行模式** `ES_MULTILINE | ES_AUTOVSCROLL` 支持自动垂直滚动
- **左侧边距** 8px 使文字不靠边
- **命令解析器** 支持完整 AutoCAD 别名（见下表）
- **光标保护** — 通过 `SetSel` + `PreTranslateMessage` 强制光标始终位于提示文字之后
- **Esc** 在命令行中取消当前操作
- **Enter** 提交命令或透传给视图完成多段线/弧

### 2. 文字输入对话框
- 用**模态输入对话框**替代硬编码的 `"CAD Text"`
- 对话框包含**文本内容**和**文字高度**两个输入字段
- 点击取消返回空闲状态，不创建实体
- 在 `LargeHW.rc` 中新增 `IDD_TEXT_INPUT` 对话框资源

### 3. 命令行坐标输入
可直接在命令行输入精确坐标：

| 格式 | 示例 | 说明 |
|------|------|------|
| `x,y` | `100,200` | 绝对世界坐标 |
| `@dx,dy` | `@50,0` | 相对上一点偏移 |
| `距离<角度` | `100<45` | 极坐标：距离 100，角度 45° |
| 纯数字 | `50` | 半径 / 角度 / 比例因子（根据当前状态） |

- 自动识别：含逗号 `,` 或 `@` 或 `<` 即进入坐标模式
- 纯数字在适当状态下自动路由到半径/角度/比例处理逻辑
- 输入的坐标同样应用网格吸附和对象捕捉

### 4. 新增绘图实体（命令行 + 菜单）
| 实体 | 菜单路径 | 命令别名 |
|------|----------|----------|
| 矩形 | 绘图 > 矩形 | `REC`、`RECTANGLE`、`RECTANG` |
| 多边形 | 绘图 > 多边形 | `POL`、`POLYGON` |
| 椭圆 | 绘图 > 椭圆 | `EL`、`ELLIPSE` |
| 文字 | 绘图 > 文字 | `T`、`TEXT`、`DT`、`DTEXT` |

### 5. 新增修改命令
| 命令 | 别名 | 操作 |
|------|------|------|
| 移动 | `M`、`MOVE` | 选择 → 基点 → 目标 |
| 复制 | `CO`、`COPY`、`CP` | 选择 → 基点 → 目标 |
| 旋转 | `RO`、`ROTATE` | 选择 → 中心 → 角度 |
| 缩放 | `SC`、`SCALE` | 选择 → 基点 → 比例 |
| 镜像 | `MI`、`MIRROR` | 选择 → 镜像线 P1 → P2 |
| 偏移 | `O`、`OFFSET` | 选择实体 → 点击偏移侧 |
| 删除 | `E`、`ERASE`、`DEL` | 删除已选中的实体 |

### 6. 视图与编辑命令
| 命令 | 别名 | 功能 |
|------|------|------|
| 撤销 | `U`、`UNDO`、Ctrl+Z | 回退上一步 |
| 重做 | `REDO`、Ctrl+Y | 恢复撤销 |
| 窗口缩放 | `Z`、`ZOOM` | 拖拽窗口放大 |
| 范围缩放 | `ZE`、`ZOOME` | 显示全部实体 |
| 平移 | `P`、`PAN` | 中键拖拽平移 |
| 切换网格 | `GRID`、F7 | 浅灰点阵网格 |
| 切换吸附 | `SNAP`、F9 | 网格吸附 |
| 切换正交 | `ORTHO`、F8 | 水平/垂直约束 |
| 切换对象捕捉 | `OSNAP`、F3 | 对象捕捉（端点/中点/圆心/象限点） |

### 7. 重做支持
- `Ctrl+Y` / `REDO` 命令恢复被撤销的操作
- 已添加到编辑菜单
- 多实体操作的完整撤销/重做栈（最多 50 步）

### 8. 对象捕捉
- `F3` 全局开关 OSNAP
- 支持端点、中点、圆心、象限点、最近点捕捉
- 激活时显示青色方块捕捉标记
- 鼠标点击和坐标输入均适用

### 9. 网格、吸附、正交的可视反馈
- 网格：深色背景上的浅灰点阵
- 吸附状态在状态栏中显示
- 正交：约束光标相对第一点水平/垂直移动
- 所有开关状态实时反映在状态栏

### 10. 夹点编辑与修改撤销
- 点击实体选中 → 显示青色夹点
- 拖拽夹点调整几何形状
- `RecordGripUndo()` / `RecordModifyUndo()` / `RecordCreateUndo()` 保证修改可撤

### 11. 鼠标中键平移
- 中键按住拖拽平移视图
- 平移中实时视觉反馈

### 12. 格式 > 图层菜单
- `ID_FORMAT_LAYER` 菜单项打开图层对话框
- 支持多个命名图层
- 状态栏显示当前图层

### 13. 右键菜单
- 右键显示：确认、取消、重复上一命令、窗口缩放、范围缩放

### 14. SCR 脚本录制与读取
- 新增 脚本 菜单：运行脚本、开始录制、停止录制
- `SCR` / `SCRIPT` 读取 `.scr` 文件并复用现有命令调度执行
- `SCRREC` / `SCRIPTREC` / `RECORDSCRIPT` 录制后续命令行输入、菜单命令和鼠标拾取坐标
- `SCRSTOP` / `SCRIPTSTOP` / `STOPSCRIPT` 停止录制并关闭脚本文件
- 脚本解析支持空行 Enter、`;` / `#` 注释，以及 `TEXT x,y height "content"` 文字输入
- 扩展 SCR 读取覆盖：`_.` / `_` / `-` 命令前缀、`LINE`、`CIRCLE`、`ARC _C`、`PLINE`、`RECTANGLE`、`ELLIPSE`、`POLYGON`、`POINT`、`LAYER`、`COLOR`、`LINETYPE`、`LINEWEIGHT`、`GRID/SNAP/ORTHO/OSNAP ON|OFF`，以及 `SELECT/ERASE/MOVE/COPY/ROTATE/SCALE/MIRROR/OFFSET ALL ...`

## Bug 修复

### 视图遮挡命令行
- **问题：** 视图占满整个客户区，命令行编辑框被完全遮挡
- **修复：** `CMainFrame::OnSize` 中通过 `GetActiveView()->MoveWindow()` 将视图底部缩小，为 60px 命令行留出空间

### 鼠标移动时命令行文字被覆盖
- **问题：** `UpdateStatusBar()` 在每次 `OnMouseMove` 中调用 `SetWindowText`，导致光标丢失、用户输入被清除
- **修复：** 将 `SetWindowText` 从 `UpdateStatusBar()` 移至 `SetDrawState()`，命令行仅在状态切换时更新

### 光标位于提示文字之前
- **问题：** 光标默认在位置 0（文字最前面），导致输入跑到提示前（如 `LCommand: `）
- **修复：** 每次程序调用 `SetWindowText` 后紧跟 `SetSel(len, len)` 将光标强制移至末尾；`PreTranslateMessage` 拦截鼠标点击并纠正光标位置

### 提示文字被当作命令输入
- **问题：** 提示前缀剥离仅当输入以提示开头时才生效，光标错位时失效
- **修复：** 改用 `Find()` 在字符串任意位置搜索提示，提取提示后的所有文本作为命令
